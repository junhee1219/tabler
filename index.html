개인정보처리방침

최종 업데이트: 2025년 4월 28일

TableExporter(이하 "당사" 또는 "확장 프로그램")는 사용자로부터 어떠한 개인정보도 수집하지 않으며, 아래와 같이 처리 정책을 안내합니다.

1. 수집하는 정보

  • 본 확장 프로그램은 사용자 개인정보(이름, 이메일, 위치, IP, 쿠키, 세션 정보 등)를 일절 수집하지 않습니다.
  • 브라우저 chrome.storage.local에 저장되는 정보는 “최근 내보내기 파일명”과 “타임스탬프” 뿐이며, 이는 로컬 영역에만 보관됩니다.

2. 정보의 이용 및 보관

저장된 최근 내보내기 기록은 사용자가 직접 팝업에서 확인·삭제할 수 있으며, 당사는 이를 외부 서버로 전송하거나 제3자와 공유하지 않습니다.

3. 사용 권한 안내

  • activeTab: 사용자가 클릭한 현재 탭에서만 테이블 추출 기능을 실행하기 위해 사용
  • scripting: 콘텐츠 스크립트를 동적으로 주입하여 테이블 선택 모드를 제공하기 위해 사용
  • downloads: 변환된 XLSX 파일을 자동으로 다운로드하기 위해 사용
  • storage: "최근 내보내기" 기록을 브라우저 로컬 스토리지에 보관하기 위해 사용
  • 호스트 권한: *://*/* 패턴으로 모든 페이지의 <table> 요소를 추출하기 위해 사용

4. 개인정보 비수집 안내

본 확장 프로그램은 로그 수집, 원격 분석 또는 사용자 식별을 위한 어떠한 추적 기능도 포함하지 않으며, 외부 서버와의 통신은 전혀 이루어지지 않습니다.
